# Clinic Management System
Clinic manager which allows for adding new patients to a database, recording notes, removing patients from the system and updating patient information using a GUI application with persistence and serialization.

To Run this program, one must have PyQt6 installed on their system.

## Instruction to run program
The first method is through a GUI application, to execute the program this way, simply run this command from inside the a5 directory:
`python -m clinic gui`

The second method is through the command-line interface (CLI).  
To run the program this way, execute the command: `python -m clinic cli` from inside the a5 directory.
